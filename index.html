< html> < /html> etiqueta Le indica al navegador que es una página web HTML. Se abre arriba de todo y se cierra abajo de todo. < html> contenido de la página web < /html> 1 5-ago-19
< head> < /head> etiqueta Le indica al navegador que acá comienza la información que debe tener sobre la página web. Se abre después de < html> y se cierra al terminar el contenido del head. < head> contenido del head < /head> 1 5-ago-19
< title> etiqueta Le indica al navegador el título de la página web (se ve arriba en la barra del navegador). Va dentro del head. < title> Mi primera página web < /title> 1 5-ago-19
< style type="text/css> etiqueta(style type) atributo(text/css) Le indica al navegador el estilo de la página web en HTML. Va dentro del head. Incluye los estilos que tienen los diferentes elementos de la página web. < style type=" text/css"> h1 { color: red; } p { color: blue; } .verde { color: green; } #titulo { color: orange; } < / style> 1 5-ago-19
